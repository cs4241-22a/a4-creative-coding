Quick Start

This is a simple Mandelbrot Set visualizer.

Here is some info on Mandelbrot sets from Wolfram Mathworld:

"The term Mandelbrot set is used to refer both to a general class of fractal sets and to a particular instance of such a set. In general, a Mandelbrot set marks the set of points in the complex plane such that the corresponding Julia set is connected and not computable."

Parameters can be changed by using the Settings panel on the top right. The following describes the various settings:

  1. iterations : Maximum number of iterations before assuming a complex point is not in the mandelbrot set
  2. escape_radius : The maximum distance from the origin, that a complex point will be evaluated
  3. mandelbrot_color: The color of the complex point which signifies the point is in the mandelbrot set
  4. Escaped Color Palette: Choose a color palette for complex points not in the mandelbrot set. Colors will be chosen based on how fast the complex point escaped.
  5. Save Image: Download the generated mandelbrot set as png

Note: Mandelbrot set generation is computationally heavy, the website may take sometime to update the display after an user event.

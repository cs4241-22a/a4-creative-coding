Data Visualization of NASA's "Near-Earth Asteroids and Comets" Dataset.

    Bar Graph:

  • X axis: Different designations for each asteroid
  • Y axis: Period years for each asteroid

    TweakPane:

    use this interface to change what data you are viewing

  • onlyPHA: click the box to also see data for potentially hazardous asteroids ( PHA)
  • maxHag: slide to see the how many asteroids are below a certain aboslute magnitude (hMag) value
  • onlyYear: pick the year which you exclusively want to see data for
  • OrbitClass: pick the orbit class that you exclusively want to see
  • "restart" restores all your filters

      Circle:

      Data: represent the amount of potentially hazardous asteroids (PHA), and non-potentially harzardous asteroids (nPHA)

    • PHA is colored red
    • nPHA is colored cyan
